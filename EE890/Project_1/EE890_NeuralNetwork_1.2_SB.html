The general format of the cable equation is given by:¶

2(,)/2 (,) + (,) = (,)/ Use the finite difference time domain algorithm to find the distribution of the membrane potential when the axon is stimulated with a temporally and locally limited narrow pulse which is modeled by: (,) = ( 1)(). Plot the snapshots of the membrane potential at = 0.1, 0.2, , 1.0. (suppose that the axon has sealed ends or = 0.0 when = +6 = 4).

Here, U is a function of space and time and we have to find the find the distribution of the membrane potential with respect to space and time when when the axon is stimulated with a temporally and locally limited narrow pulse which is modeled by: (,) = ( 1)().

We are given that i(,) = ( 1)(), that means when x=1 and t=0, (,) = 1 otherwise 0 for t=0. First we can solve the Stationary solution of the cable equation with respect to t=0 and only space or x can change using finite difference method. Once we have found the distribution of the membrane potential for every node at t=0, we can calculate distribution of the membrane potential for t=0.1 t0 1 at each node using Finite Difference Method in Time Domain. as t is constant, (,)/ =0 and at t=0 the equation becomes 2(,)/2 (,)= -(,)

The finite difference method is a means of solving differential equations numerically by using a finite value for delta x rather than delta x -> 0.

The general equation 2nd order differential equation in terms of finite difference method given by:

𝜕2𝑢(𝑥,𝑡)/𝜕𝑥2 = [u(x(i+1),t) - 2u(x(i),t) + u(x(i-1),t) ] / (delta x)^2¶

So we can break our given 2nd order differential equation into a bunch of linear equations of u to find out the value of u at different nodes. For the shake of simplicity I am taking the number of nodes =11 and delta x=1 and i ={1, ..., 11}

|----|----|----|----|----|----|----|----|----|----|
-4   -3   -3   -1   0    1    2    3    4    5    6

Now we will write the linear equation of u at each node. We already know the boundary conditions. So, U1=U(-4,t=0)=0 and U11=U(6,t=0)=0 Now,

i=2, U3 - 3U2 - U1 = 0
i=3, U4 - 3U3 - U2 = 0
i=4, U5 - 3U4 - U3 = 0
i=5, U6 - 3U5 - U4 = 0
i=6, U7 - 3U6 - U5 = -1 [because i(,) = (  1)()]
i=7, U8 - 3U7 - U6 = 0
i=8, U9 - 3U8 - U7 = 0
i=9, U10 - 3U9 - U8 = 0
i=9, U11 - 3U10 - U9 = 0

As, we have constructed our A.x =b format, we can use Steepeset Descent to solve for u at different points. Once we get the distribution of the membrane potential for every node at t=0, we will transfer those values to a NTimes x NSpace Matrix as 0th row. Then iteratively we will calculate the distribution of the membrane potential for every node at t=0.1 to 1 gradually using Finite Difference in Time Domain Algorithm.

Solution for T at the next (future) time step j for node i is: T(i,j)=T(i-1,j) + K* (deltaT/deltaX^2)[T(i-1,j-1) - 2T(i-1,j)+ T(i-1,j+1)]

Steepest Descent:¶

If we have an equation in A.x=b form, where A is N x N matrix and b, x are N dimensional vectors, Steepest Descent method helps to find the solution for x in a iterative process by implementing line search mechanism repeatedly. In the method of Steepest Descent, we start at an arbitrary point in N dimension, x(0), then we take each step in the direction(residual at that point) where the f decreases most quickly, until we are satisfied that we are close enough to the solution of x. Alpha() or step-size in a line search procedure tells us where in the line or direction of the movement the value of f will be minimized. The residual of the current step and the direction of the movement to be taken from the current point is orthogonal to each other. The next value of x is calculated with the Linear Combination of current value of x, Step Size or Alpha and Residual at that point.

iteration=100000   # Maximum Number of Iterations (Arbritary) 
r=b-np.dot(A,x)    # Initial Residual
delta= np.dot(r.T, r)[0,0] # Norm of Residual (How far from the solution).
delta0=delta # Make a copy od Initial Residual Norm
i=0
while i<iteration and delta > 0.000001*delta0 : #Loop will run if current iteration< Max iteration and current residual is above 
                                                #tolerance level
    q=np.dot(A,r)                       # Calculating A*r, which will be used for Alpha and Next r or residual Calculation
    alpha= delta/np.dot(r.T,q)[0,0]     # Calculating Alpha or Stepsize. It is a Scaler.
    x=x+(alpha*r)                       # Calculating next x, by taking alpha stepsize in the direction of r or current residual.
    if (i%50)==0:
        r=b-np.dot(A,x)                 # Number 50 is arbitrary and this step is to reduce appox. error.
    else:
        r=r-(alpha*q)                   # Calculating the new residual, r= r(cur) - (step_size*(A*r(cur)))
    delta=np.dot(r.T, r)[0,0]           # Norm of Just Calculated Residual (How far from the solution).
    i=i+1                               # increase Counter
